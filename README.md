## General info
Simple graphql demo using laravel

## Technologies
Project is created with:
* PHP 8
* Docker 20.10.6
* Docker-Compose 1.29.1
* Laravel 8
* Composer 2.0.12

## Setup
Make sure to have all the dependencies above installed locally and docker running in the background.

```
$ composer install
$ ./vendor/bin/sail up -d 
```
